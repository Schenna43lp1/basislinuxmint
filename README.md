# Markus Linux Setup

Automatisches Setup für meine Linux Workstation.

## Installation

Repo klonen

git clone https://github.com/USERNAME/linux-setup.git

Script starten

chmod +x setup-markus.sh
./setup-markus.sh

## Features

- Dev Tools
- Docker
- Networking Tools
- Monitoring Tools
- Homelab Ordnerstruktur
- ZSH + Starship
- Custom Aliases

## Struktur

dev/        → Coding Projekte  
homelab/    → Proxmox / Infrastruktur  
docker/     → Container Projekte  
monitoring/ → Uptime / Metrics  
scripts/    → Automationen  
trading/    → Trading Tools
